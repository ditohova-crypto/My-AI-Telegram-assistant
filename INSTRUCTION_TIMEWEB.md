## 🐾 Шаг 1. Регистрация на Timeweb Cloud

1. Откройте **https://cloud.timeweb.com**
2. Нажмите **"Регистрация"**
3. Введите:
   - **Email**
   - **Пароль** (сложный)
   - **Номер телефона** (Россия)
4. Получите SMS-код, введите
5. Подтвердите email (перейдите по ссылке из письма)
6. **Готово!** Вы в панели управления.

---

## 💳 Шаг 2. Пополнение баланса

1. В панели Timeweb нажмите **"Баланс"** (вверху)
2. Нажмите **"Пополнить"**
3. Выберите способ:
   - Банковская карта
   - ЮMoney
   - Система быстрых платежей (СБП)
   - Криптовалюта
4. Введите сумму **500₽** (с запасом на первый месяц)
5. Оплатите

> ⚠️ Баланс должен быть положительным, иначе сервер заблокируют через 3 дня.

---

## 🖥️ Шаг 3. Создание сервера

1. В панели нажмите **"Создать сервер"** (большая зелёная кнопка)
2. Выберите конфигурацию:

| Параметр | Значение |
|----------|----------|
| **Тариф** | **"Облачный VPS"** |
| **Операционная система** | **Ubuntu 22.04 LTS** |
| **Процессор** | 1 vCPU |
| **RAM** | 1 ГБ |
| **Диск** | 15 ГБ NVMe |
| **Локация** | Москва (или ближайшая) |

3. Нажмите **"Создать"**
4. Ждите 1-2 минуты — сервер создаётся

> 💡 Можно взять тариф за 200₽ (512MB RAM), но 1GB надёжнее.

## 🔑 Шаг 4. Подключение к серверу

### Вариант А: Через браузер (проще)
1. В панели Timeweb нажмите на ваш сервер
2. Нажмите **"Веб-консоль"** или **"VNC"**
3. Откроется чёрное окно терминала прямо в браузере

### Вариант Б: Через SSH (если удобно)
1. Timeweb покажет **IP-адрес** сервера (например `89.108.XX.XX`)
2. Покажет **пароль root** (скопируйте его!)
3. Откройте терминал на компьютере:
```bash
ssh root@89.108.XX.XX
# введите пароль root (который показал Timeweb)
```

> 💡 Рекомендую Вариант А (веб-консоль) — не нужно ничего устанавливать на компьютер.

## 🤖 Шаг 5. Установка бота (копировать и вставлять)

После подключения вы увидите:
```
root@ubuntu:~$
```

### 5.1 Обновить систему
```bash
apt update && apt upgrade -y
```
Ждите 2-3 минуты.

### 5.2 Установить Python
```bash
apt install -y python3.11 python3.11-venv python3-pip git curl nano
```

### 5.3 Создать папку для бота
```bash
mkdir -p /opt/telegram-bot
cd /opt/telegram-bot
```

### 5.4 Создать виртуальное окружение
```bash
python3.11 -m venv venv
source venv/bin/activate
```

Появится `(venv)` слева от команд — значит всё ок.

### 5.5 Загрузить код
Если у вас есть GitHub:
```bash
git clone https://github.com/ВАШ_НИК/telegram-ai-bot.git /tmp/bot-files
cp /tmp/bot-files/*.py /tmp/bot-files/requirements.txt /tmp/bot-files/.gitignore .
```

Если **нет GitHub** — создадим файлы вручную:
```bash
nano requirements.txt
```
Откроется редактор. Вставьте:
```
python-telegram-bot>=20.7
openai>=1.0.0
python-dotenv>=1.0.0
apscheduler>=3.10.4
requests>=2.31.0
```
Нажмите **Ctrl+O** → **Enter** → **Ctrl+X** (сохранить и выйти).

### 5.6 Установить библиотеки
```bash
pip install -r requirements.txt
```
Ждите 2-3 минуты.

## ⚙️ Шаг 8. Создать системный сервис (автозапуск)

Это важно — чтобы бот запускался сам после перезагрузки.

```bash
nano /etc/systemd/system/telegram-bot.service
```

Вставьте:
```ini
[Unit]
Description=AI Telegram Bot
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/telegram-bot
Environment=PATH=/opt/telegram-bot/venv/bin
EnvironmentFile=/opt/telegram-bot/.env
ExecStart=/opt/telegram-bot/venv/bin/python /opt/telegram-bot/bot_vps.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

Сохраните: Ctrl+O → Enter → Ctrl+X

Активируйте:
```bash
systemctl daemon-reload
systemctl enable telegram-bot
systemctl start telegram-bot
```

## ✅ Шаг 9. Проверить что работает

### Статус:
```bash

```
Должно быть зелёное **"active (running)"**.

### Логи в реальном времени:
```bash

```
Нажмите **Ctrl+C** чтобы выйти из просмотра логов.

### Если ошибка:
```bash

```
Скопируйте последние 5 строк и пришлите мне — исправим.

## 🆘 Если что-то не работает

### "active (failed)" в статусе
```bash
journalctl -u telegram-bot -n 30
```
Скопируйте ошибку мне.

### Бот молчит
1. Проверьте `.env` — правильный ли токен и API-ключ?
2. Пополнен ли баланс Moonshot?
3. Пишите в логи — `journalctl -u telegram-bot -f` и пишите боту — видите ли активность?

### Напоминания не приходят
Проверьте время на сервере:
```bash
date
```
Сервер может быть по UTC (на 3 часа раньше Москвы). Напоминания настроены на серверное время.
