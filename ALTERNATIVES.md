# 🔑 Где взять API-ключ (альтернативы Moonshot AI)

Moonshot AI — китайская платформа, и регистрация там часто требует китайский номер телефона или WeChat. **Не переживайте** — код бота поддерживает **любой OpenAI-compatible API**, и есть отличные альтернативы, которые работают из любой точки мира.

---

## ⭐ Рекомендуемый вариант: Groq (самый простой)

**Почему Groq:**
- ✅ Регистрация за 1 минуту через Google/GitHub
- ✅ API-ключ выдаётся мгновенно
- ✅ Бесплатный тариф: **$5/месяц** + 20 запросов/мин
- ✅ Полностью совместим с кодом (тот же OpenAI SDK)
- ✅ Работает из любой страны
- ✅ Очень быстрые ответы (модели Llama, Mixtral)

### Как получить ключ (пошагово):

1. Перейдите на **https://console.groq.com**
2. Нажмите **"Sign Up"** → зарегистрируйтесь через **Google** или **GitHub**
3. После входа нажмите **"API Keys"** в боковом меню
4. Нажмите **"Create API Key"**
5. Скопируйте ключ (начинается с `gsk_...`)
6. Готово! Ключ можно использовать сразу.

### Настройка .env:
```env
AI_API_KEY=gsk_your_real_key_here
AI_BASE_URL=https://api.groq.com/openai/v1
AI_MODEL=llama-3.3-70b-versatile
```

### Доступные модели Groq (бесплатный tier):
| Модель | Описание |
|--------|----------|
| `llama-3.3-70b-versatile` | Универсальная, умная, для сложных задач |
| `llama-3.1-8b-instant` | Быстрая, для простых вопросов |
| `mixtral-8x7b-32768` | Хорошая для длинных текстов (32K контекст) |
| `gemma2-9b-it` | Лёгкая и быстрая |

## 🔄 Альтернатива: OpenRouter (бесплатные модели)

**Почему OpenRouter:**
- ✅ Регистрация через email/Google
- ✅ Доступ к **бесплатным** моделям (DeepSeek, Llama и др.)
- ✅ Один ключ — доступ к 100+ моделям
- ✅ Можно использовать без привязки карты

### Как получить ключ:

1. Перейдите на **https://openrouter.ai**
2. Нажмите **"Sign In"** → зарегистрируйтесь
3. Перейдите в **"Keys"** (https://openrouter.ai/keys)
4. Создайте ключ, скопируйте (начинается с `sk-or-v1-...`)

### Настройка .env:
```env
AI_API_KEY=sk-or-v1-your_real_key_here
AI_BASE_URL=https://openrouter.ai/api/v1
AI_MODEL=deepseek/deepseek-chat:free
```

> ⚠️ Бесплатные модели на OpenRouter иногда медленные или недоступны при высокой нагрузке. Если бот долго не отвечает — попробуйте другую модель.

## 💳 Альтернатива: OpenAI (ChatGPT)

**Если у вас есть карта Visa/MasterCard:**

1. Перейдите на **https://platform.openai.com**
2. Зарегистрируйтесь
3. Перейдите в **"API Keys"** → создайте ключ
4. Пополните баланс минимум на $5

### Настройка .env:
```env
AI_API_KEY=sk-your_openai_key_here
AI_BASE_URL=https://api.openai.com/v1
AI_MODEL=gpt-4o-mini
```

---

## 🆓 Альтернатива: Google Gemini (бесплатный tier)

1. Перейдите на **https://aistudio.google.com/app/apikey**
2. Войдите через Google-аккаунт
3. Нажмите **"Create API Key"**
4. Скопируйте ключ

### Настройка .env:
```env
AI_API_KEY=your_gemini_key_here
AI_BASE_URL=https://generativelanguage.googleapis.com/v1beta/openai/
AI_MODEL=gemini-1.5-flash
```

## 🖥️ Локальный AI (без интернета)

Если у вас мощный компьютер, можно запустить модель локально через **LM Studio** или **Ollama**:

```env
AI_API_KEY=not-needed
AI_BASE_URL=http://localhost:1234/v1
AI_MODEL=local-model-name
```

## ❗ Если ничего не работает

1. **Проверьте ключ**: вставьте его в `.env` без лишних пробелов и кавычек
2. **Проверьте base_url**: должен заканчиваться на `/v1` (например `https://api.groq.com/openai/v1`)
3. **Проверьте баланс**: у Groq бесплатный tier автоматический, у OpenRouter — проверьте лимиты
4. **Посмотрите логи**: при запуске бота в консоли будет точная ошибка

---

**Мой совет:** начните с **Groq**. Регистрация занимает 1 минуту, бесплатный тариф щедрый, а модели Llama 3.3 работают с русским языком очень хорошо.
