# 🤖 AI Telegram Bot

Полноценный Telegram-ассистент с памятью диалогов (SQLite), системными инструкциями и поддержкой любого OpenAI-compatible API. Работает с **Moonshot AI (Kimi)**, Groq, OpenRouter, OpenAI и другими.

---

## 📋 Возможности

- 💬 **Интеллектуальные диалоги** через современные LLM
- 🧠 **Контекстная память** — SQLite хранит до 20 последних сообщений
- 🎯 **Системные инструкции** — настраиваемый промпт
- 📊 **Статистика** — подсчёт сообщений и токенов
- 🔐 **Админ-панель** — команды только для владельца
- 🛡️ **Обработка ошибок** — защита от сбоев API, лимитов, длинных сообщений
- 📝 **Логирование** — подробные логи в файл и консоль
- ☁️ **Готовый деплой** — Docker + инструкции для Render, Railway, Fly.io

## 🚀 Быстрый старт

### 1. Сохраните файлы

Скачайте все файлы в одну папку. Структура должна быть такой:

```
kimi-bot/
├── .env              ← ваши секреты (ключи, токены)
├── .env.example      ← шаблон для других
├── .gitignore        ← защита от случайной публикации .env
├── bot.py            ← основной файл (локальный запуск)
├── main_webhook.py   ← для облака (Render и др.)
├── config.py         ← настройки (читаем из .env)
├── database.py       ← SQLite: история, пользователи
├── Dockerfile        ← сборка контейнера
├── requirements.txt  ← Python-зависимости
├── README.md         ← этот файл
├── DEPLOY_RENDER.md  ← деплой на Render
└── ALTERNATIVES.md   ← альтернативные AI-провайдеры
```

### 2. Заполните `.env`

Файл `.env` уже создан с вашими настройками. Проверьте его содержимое:

```env
TELEGRAM_BOT_TOKEN=8793906279:AAF6i3PeOxBmGC2fbw3Xu9WuDnbr8wpHNe8
AI_API_KEY=sk-viUUKlwRVJb9Esmz73cK7DlhikPqXlRtbwl3fCOZC3goODGx
AI_BASE_URL=https://api.moonshot.ai/v1
AI_MODEL=kimi-k2.5
AI_TEMPERATURE=1
AI_MAX_TOKENS=2048
MAX_HISTORY_MESSAGES=20
```

> ⚠️ **Безопасность**: файл `.env` уже добавлен в `.gitignore` и **никогда не попадёт на GitHub**.

### 3. Установите Python и зависимости

Требуется **Python 3.9+**.

```bash
# Проверьте версию
python --version

# Установите зависимости
pip install -r requirements.txt
```

### 4. Запустите бота

```bash
python bot.py
```

При успешном запуске увидите:
```
=== Запуск AI Telegram Bot ===
База данных инициализирована
AI клиент инициализирован | base_url=https://api.moonshot.ai/v1 | model=kimi-k2.5
Бот инициализирован. Запуск polling...
Запуск polling...
```

Напишите боту в Telegram `/start` — он должен ответить.

## 🔐 Безопасность при публикации на GitHub

Ключи и токены **не должны попасть** в публичный репозиторий.

| Файл | Содержит секреты? | Попадает в Git? |
|------|-------------------|-----------------|
| `.env` | ✅ Да (ключи, токены) | ❌ Нет (в `.gitignore`) |
| `.env.example` | ❌ Нет (шаблон) | ✅ Да |
| `config.py` | ❌ Нет (читает из `.env`) | ✅ Да |

**Если случайно опубликовали `.env`:**
```bash
git rm --cached .env        # удалить из индекса Git
git commit -m "Remove .env"
# Затем срочно отзовите токены/ключи и создайте новые!
```

Чтобы бот работал постоянно, даже когда компьютер выключен:

### Вариант A: Render.com (бесплатно)

Подробная инструкция в **`DEPLOY_RENDER.md`**.

Кратко:
1. Загрузите код на [GitHub](https://github.com) (без `.env`!)
2. Зарегистрируйтесь на [Render](https://render.com) через GitHub
3. Создайте **Web Service** (Runtime: Docker)
4. В Render добавьте **Environment Variables** — те же значения, что в `.env`
5. Установите webhook в Telegram
6. Готово — бот работает 24/7

## 🛠️ Кастомизация

### Смена модели

В `.env`:
```env
AI_MODEL=moonshot-v1-8k       # быстрая, дешёвая
AI_MODEL=moonshot-v1-32k      # для длинных текстов
AI_MODEL=kimi-k2.5             # умная (по умолчанию)
AI_MODEL=kimi-k2.6             # самая новая
```

> Для `kimi-k2.5` и `kimi-k2.6` параметр `AI_TEMPERATURE` должен быть `1` (требование API).
> Для `moonshot-v1-*` можно использовать `0.7`.

### Изменение системного промпта

В `.env`:
```env
SYSTEM_PROMPT=Ты — строгий senior-разработчик. Отвечай кратко, только код.
```

### Длина истории

```env
MAX_HISTORY_MESSAGES=10  # меньше = экономия, больше = лучше контекст
```

## 🆘 Решение проблем

| Проблема | Решение |
|----------|---------|
| `AI_API_KEY не указан` | Проверьте, что файл `.env` существует и заполнен |
| `invalid_authentication_error` | Неверный ключ — проверьте на platform.moonshot.ai |
| `insufficient balance` | Пополните баланс (минимум $1) |
| `invalid temperature` | Для kimi-k2.5/K2.6 установите `AI_TEMPERATURE=1` |
| Бот молчит | Проверьте логи в `logs/bot.log` |
