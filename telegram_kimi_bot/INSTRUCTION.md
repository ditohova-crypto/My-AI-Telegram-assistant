## 🐙 Шаг 1. Создать GitHub

Это бесплатный сайт для хранения файлов. Там будут лежать коды бота.

### 1.1 Регистрация
1. Откройте [github.com/signup](https://github.com/signup)
2. Введите вашу почту
3. Придумайте пароль
4. Введите имя пользователя (например `anastasia-bot`)
5. Подтвердите капчу ("Я не робот")
6. Нажмите **"Create account"**
7. Проверьте почту — перейдите по ссылке для подтверждения

### 1.2 Создать хранилище (репозиторий)
1. Зайдите на [github.com](https://github.com) (войдите)
2. Нажмите зелёную кнопку **"+"** (справа вверху) → **"New repository"**
3. Заполните поля:
   - **Repository name**: `telegram-ai-bot`
   - **Description**: `My AI Telegram assistant`
   - Выберите **Public** (публичный)
   - **☑️ Add a README** — галочку НЕ ставьте
   - **☑️ Add .gitignore** — галочку НЕ ставьте
   - **☑️ Choose a license** — галочку НЕ ставьте
4. Нажмите **"Create repository"**

### 1.3 Загрузить файлы
1. На странице репозитория нажмите **"uploading an existing file"** (ссылка мелким шрифтом под заголовком "Quick setup")
2. Откроется страница с надписью **"Drag files here"**
3. Откройте папку с файлами бота на вашем компьютере
4. **Важно! Удалите файл `.env` из этой папки** (переместите его на рабочий стол или другое место)
5. Выделите все оставшиеся файлы в папке (Ctrl+A)
6. Перетащите их мышкой в окно браузера на страницу GitHub
7. Дождитесь загрузки — файлы появятся в списке
8. Прокрутите вниз
9. В поле **"Commit changes"** напишите: `Initial commit`
10. Нажмите зелёную кнопку **"Commit changes"**

### 1.4 Проверить
- Зайдите на `github.com/ВАШЕ_ИМЯ/telegram-ai-bot`
- Убедитесь что **`.env` нет** в списке файлов
- Убедитесь что **`.env.example` есть** — это нормально, он без секретов

> ✅ Готово! Код бота теперь на GitHub.

## ☁️ Шаг 2. Создать Render

Это бесплатный сервер, который будет запускать вашего бота 24 часа в сутки.

### 2.1 Регистрация
1. Откройте [render.com](https://render.com)
2. Нажмите **"Get Started for Free"**
3. Нажмите **"Sign up with GitHub"** (регистрация через GitHub)
4. Разрешите доступ — нажмите зелёную кнопку **"Authorize render"**
5. Заполните форму:
   - **Username**: что угодно (например `anastasia-bot`)
   - Нажмите **"Create Account"**

### 2.2 Создать сервис
1. В панели Render нажмите синюю кнопку **"New +"** (справа вверху)
2. Выберите **"Web Service"**
3. Найдите в списке ваш репозиторий `telegram-ai-bot`
4. Нажмите **"Connect"** (серую кнопку справа)

### 2.3 Настроить сервис
Заполните поля:

| Поле | Что ввести |
|------|-----------|
| **Name** | `telegram-ai-bot` |
| **Region** | `Frankfurt (EU Central)` |
| **Runtime** | Выберите `Docker` (выпадающий список) |
| **Branch** | `main` |
| **Root Directory** | оставьте пустым |

Прокрутите вниз.

В разделе **"Instance Type"** выберите **"Free"** (бесплатный).

Нажмите **"Advanced"** (кнопка слева внизу, над "Create Web Service").

Появится раздел **"Environment Variables"** (переменные окружения).

Нажмите **"Add Environment Variable"** и добавьте 7 строк **точно** как в таблице ниже:

| Переменная (Key) | Значение (Value) |
|-----------------|------------------|
| `TELEGRAM_BOT_TOKEN` | `8793906279:AAF6i3PeOxBmGC2fbw3Xu9WuDnbr8wpHNe8` |
| `AI_API_KEY` | `sk-viUUKlwRVJb9Esmz73cK7DlhikPqXlRtbwl3fCOZC3goODGx` |
| `AI_BASE_URL` | `https://api.moonshot.ai/v1` |
| `AI_MODEL` | `kimi-k2.5` |
| `AI_TEMPERATURE` | `1` |
| `AI_MAX_TOKENS` | `2048` |
| `WEBHOOK_SECRET` | `my-secret-2024-x7k9` (придумайте любую фразу) |

> 💡 **WEBHOOK_SECRET** — это пароль. Придумайте что-то типа `anastasia-bot-secret-123`.

После заполнения нажмите **"Create Web Service"**.

Render начнёт сборку. Подождите 3-5 минут.

## ❗ Если что-то не работает

### Бот молчит
1. Зайдите в Render → ваш сервис → вкладка **"Logs"** (сверху)
2. Посмотрите последние строки — там будет ошибка
3. Если видите `insufficient balance` — пополните баланс на [platform.moonshot.ai](https://platform.moonshot.ai) (минимум $1)

### Ошибка "Unauthorized" или "401"
- WEBHOOK_SECRET в Render и в ссылке webhook должны быть **одинаковыми**
- Переустановите webhook с правильным секретом

### Сервер падает
- Проверьте в Render → Logs → ищите `AI_API_KEY` или `TELEGRAM_BOT_TOKEN`
- Если написано "не указан" — проверьте Environment Variables (Шаг 2.3)

### Webhook не устанавливается
- Удалите старый webhook:
```
https://api.telegram.org/bot8793906279:AAF6i3PeOxBmGC2fbw3Xu9WuDnbr8wpHNe8/deleteWebhook
```
- Подождите 10 секунд
- Установите заново (Шаг 3)

### Бот отвечал, а потом перестал
- На бесплатном Render сервер "засыпает" после 15 минут без сообщений
- **Это нормально!** Отправьте боту сообщение — он проснётся за 10-30 секунд
